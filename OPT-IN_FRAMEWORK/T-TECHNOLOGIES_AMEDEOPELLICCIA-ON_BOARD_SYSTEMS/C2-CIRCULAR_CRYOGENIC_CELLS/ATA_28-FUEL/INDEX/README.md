# INDEX — ATA 28 Fuel System (C2 Circular Cryogenic Cells)

This directory is the **system index** for the AMPEL360 Q100 ATA 28 Fuel subsystem.

## Contents

| File | Purpose |
|------|---------|
| `SSOT_INDEX.yaml` | Registry of authoritative source locations for every lifecycle phase and artifact type. This is the single source of truth for "where things live." |
| `TRACE_MASTER.csv` | Master traceability matrix linking requirements, design, test, and publication artifacts across lifecycle phases. |
| `ID_REGISTRY.csv` | Canonical registry of all identifiers (SSOT, REQ, KNOT, KNU, ECR, DM) allocated within ATA 28 C2 scope. |
| `ARTIFACT_CATALOG.yaml` | Inventory of artifact types with counts, locations, and naming patterns. |
| `NAVIGATION_MAP.html` | **(generated)** Auto-generated HTML navigation map — do not edit manually. |

## Usage

1. Start with `SSOT_INDEX.yaml` to locate authoritative sources.
2. Use `TRACE_MASTER.csv` to verify traceability coverage.
3. Consult `ID_REGISTRY.csv` before allocating new identifiers.
4. Review `ARTIFACT_CATALOG.yaml` for artifact type metadata.
