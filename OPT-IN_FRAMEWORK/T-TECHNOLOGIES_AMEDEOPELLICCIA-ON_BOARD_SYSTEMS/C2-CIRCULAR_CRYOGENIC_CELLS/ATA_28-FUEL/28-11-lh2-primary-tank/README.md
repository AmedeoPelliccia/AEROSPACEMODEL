# 28-11 — LH₂ Primary Tank (LH₂ Circular Cryogenic Cells)

**ATA Section:** 28-11

## Scope

This section covers the **lh₂ primary storage tank** subsystem for liquid hydrogen (LH₂) circular cryogenic cells:

- Primary cryogenic storage tank design and certification
- Vacuum-jacketed tank construction and materials
- Tank structural analysis and pressure vessel qualification
- Primary tank insulation system (MLI)
