# IDB — ATA 28-10-00 Fuel Storage General

## Information Deliverable Database

The IDB contains all deliverable artifacts for the C2 Circular Cryogenic
Cells fuel storage system (ATA 28-10-00).

## Structure

| Directory | Purpose |
|-----------|---------|
| `OPS/` | Operational in-service lifecycle data (LC11–LC14) |
| `PUB/` | Publication deliverables — S1000D CSDB, exports, IETP |
| `INDEX/` | Release notes, trace summaries, manifests, compliance |

## OPS — Operational Lifecycle

- **LC11** — Operations customization and operator configuration
- **LC12** — Support services (SBs, repairs, queries, COC)
- **LC13** — MRO sustainment (maintenance, repair, overhaul)
- **LC14** — Retirement, dismantling, circularity, DPP closure

## PUB — Publications

`PUB/` is generated by ASIT from KDB sources via approved contracts.
Each publication type has three output tiers:

- `CSDB/` — S1000D data modules (BREX-validated)
- `EXPORT/` — Rendered PDF and HTML
- `IETP_RUNTIME/` — Interactive viewer packages
