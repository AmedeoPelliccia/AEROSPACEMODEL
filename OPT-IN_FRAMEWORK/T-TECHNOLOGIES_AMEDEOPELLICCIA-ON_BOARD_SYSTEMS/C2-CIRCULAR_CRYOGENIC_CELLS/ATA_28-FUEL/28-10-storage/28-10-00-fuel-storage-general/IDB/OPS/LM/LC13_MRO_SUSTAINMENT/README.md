## ATA 28-10-00 Fuel Storage General

Maintenance, repair, and overhaul sustainment for the C2 cryogenic fuel
storage system. Content from three source packages publishes to IDB/PUB/:

| Source Package | Publication Target | Description |
|----------------|--------------------|-------------|
| `Maintenance_Source/` | `IDB/PUB/AMM/` | Scheduled maintenance, task cards |
| `Repair_Source/` | `IDB/PUB/SRM/` | Structural repairs, damage disposition |
| `Overhaul_Source/` | `IDB/PUB/CMM/` | Component overhaul, shop visits |

All MRO data is subject to BREX validation and hydrogen/cryogenic safety
checks before publication.
