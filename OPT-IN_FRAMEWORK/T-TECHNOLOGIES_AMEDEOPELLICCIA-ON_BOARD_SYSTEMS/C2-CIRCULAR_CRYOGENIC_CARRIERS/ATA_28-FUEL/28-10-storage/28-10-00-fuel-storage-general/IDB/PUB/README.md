# PUB — Publications

## ATA 28-10-00 Fuel Storage General

All publication deliverables for the C2 cryogenic fuel storage system.

## Structure

| Publication | Source | Description |
|-------------|--------|-------------|
| `AMM/` | `LC13/Maintenance_Source` | Aircraft Maintenance Manual |
| `SRM/` | `LC13/Repair_Source` | Structural Repair Manual |
| `CMM/` | `LC13/Overhaul_Source` | Component Maintenance Manual |
| `IPC/` | Parts data | Illustrated Parts Catalogue |

## Output Tiers

Each publication type contains three output tiers:

- **`CSDB/`** — S1000D data modules (BREX-validated, schema-compliant)
- **`EXPORT/`** — Rendered deliverables in PDF and HTML format
- **`IETP_RUNTIME/`** — Interactive Electronic Technical Publication viewer

All content is governed by BREX rules and generated via ASIT contracts.
