# ATA 45 — Central Maintenance System

**Domain:** T-TECHNOLOGIES / D-DATA
**Boundary:** SHARED
**Doc ID:** T-D-ATA45-001
**Lifecycle:** Standard (LC01–LC13)

---

## Scope

Centralized fault detection, isolation, and maintenance support system for the AMPEL360 Q100. Aggregates Built-In Test Equipment (BITE) data from all aircraft systems. Provides ground maintenance terminal interface and health monitoring.
