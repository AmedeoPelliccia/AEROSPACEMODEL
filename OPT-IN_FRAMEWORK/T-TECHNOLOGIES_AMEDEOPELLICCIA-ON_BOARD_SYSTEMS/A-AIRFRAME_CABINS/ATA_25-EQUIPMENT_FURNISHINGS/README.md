**Domain:** T-TECHNOLOGIES / A-AIRFRAME_CABINS

**Doc ID:** T-A-ATA25-001
**Lifecycle:** Standard (LC01–LC13)

## Scope

Covers interior equipment, passenger seats, galleys, lavatories, and cabin furnishings for the AMPEL360 Q100. Defines structural attachment points and load requirements. Programme-specific cabin layouts are maintained in programme forks.
