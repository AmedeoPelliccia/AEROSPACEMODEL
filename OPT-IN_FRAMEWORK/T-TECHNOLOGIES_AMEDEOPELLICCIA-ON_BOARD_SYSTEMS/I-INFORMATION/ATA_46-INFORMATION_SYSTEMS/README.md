# ATA 46 — Information Systems

**Domain:** T-TECHNOLOGIES / I-INFORMATION
**Boundary:** SHARED
**Doc ID:** T-I-ATA46-001
**Lifecycle:** Standard (LC01–LC13)

## Scope

Aircraft information management systems for the AMPEL360 Q100 including electronic flight bag integration, aeronautical databases, and crew information services. Manages navigation database updates and electronic documentation. Interfaces with ATA 34 navigation and ATA 45 maintenance systems.
