# ATA 23 — Communications

**Domain:** T-TECHNOLOGIES / C1-COMMS
**Boundary:** SHARED
**Doc ID:** T-C1-ATA23-001
**Lifecycle:** Standard (LC01–LC13)

---

## Scope

Aircraft communication systems for the AMPEL360 Q100 including VHF/HF radio, ACARS, SATCOM, and cockpit voice recorder. Defines communication bus architecture and antenna locations. Interfaces with ATA 34 navigation and ATA 46 information systems.
