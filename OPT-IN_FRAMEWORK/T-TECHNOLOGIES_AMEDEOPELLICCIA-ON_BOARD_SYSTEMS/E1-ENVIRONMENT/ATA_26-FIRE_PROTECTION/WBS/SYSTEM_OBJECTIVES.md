# ATA 26 – System Objectives

**System:** AMPEL360 Q100 Fire Protection System (E1 Environment)
**ATA Chapter:** 26 – Fire Protection
**Version:** 1.0.0

## Measurable Objectives

### Performance

| ID      | Objective                        | Target        | Verification |
|---------|----------------------------------|---------------|--------------|
| OBJ-01  | Detection response time          | < 5 s         | Test         |
| OBJ-02  | False alarm rate                 | < 0.01%/FH    | Analysis     |
| OBJ-03  | Suppression discharge time       | < 1 s         | Test         |
| OBJ-04  | H₂ detection threshold           | 0.1% LEL      | Test         |
| OBJ-05  | Detection system reliability     | > 0.999       | Analysis     |

### Programme

| ID      | Objective                        | Target        | Verification |
|---------|----------------------------------|---------------|--------------|
| OBJ-06  | Certification completion         | Q4 2027       | Review       |
| OBJ-07  | Requirement traceability         | 100%          | Inspection   |
| OBJ-08  | Unresolved safety KNOTs at CDR   | 0             | Review       |

### Safety

| ID      | Objective                        | Target              | Verification |
|---------|----------------------------------|----------------------|--------------|
| OBJ-09  | Undetected fire probability     | < 1E-9 per FH       | Analysis     |
| OBJ-10  | Suppression system failure      | < 1E-7 per FH       | Analysis     |
| OBJ-11  | H₂ fire prevention              | < 1E-9 per FH       | Analysis     |

## Traceability

Each objective traces to one or more WBS work packages and is verified at
the designated lifecycle gate (PDR, CDR, TRR, or FRR).
