# P-PROGRAMS Index

## Subdomain Structure

|-----------|------|--------------|-------------|

---

## P — Product Definition

Defines the aircraft's physical characteristics and identification on the ground.
Maps to configuration management workflows.

| ATA | Directory | Description |
|-----|-----------|-------------|
| 06 | `ATA_06-DIMENSIONS_AND_AREAS/` | Dimensions, areas, and zonal arrangements |
| 08 | `ATA_08-LEVELING_AND_WEIGHING/` | Leveling and weighing operations |
| 11 | `ATA_11-PLACARDS_AND_MARKINGS/` | Placards and marking requirements |

## S — Service Instruction

Covers ground crew interaction procedures for aircraft handling and servicing.
Maps to ground operations manual workflows.

| ATA | Directory | Description |
|-----|-----------|-------------|
| 07 | `ATA_07-LIFTING_AND_SHORING/` | Jacking, lifting, and shoring procedures |
| 09 | `ATA_09-TOWING_AND_TAXIING/` | Towing and taxi procedures |
| 10 | `ATA_10-PARKING_MOORING_STORAGE_RETURN_TO_SERVICE/` | Parking, storage, and return to service |
| 12 | `ATA_12-SERVICING/` | Routine servicing procedures |
