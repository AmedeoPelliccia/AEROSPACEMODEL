## Purpose

Defines the specification for the automated compliance dashboard.

## Required Views

1. **Product Compliance Summary** — Overall compliance percentage by product/variant
2. **Phase Compliance Matrix** — Compliance status per lifecycle phase
3. **KNOT Status Distribution** — Count and percentage of KNOT status codes
4. **Evidence Completeness** — Evidence coverage vs. requirements
5. **Orphan Detection** — List of orphan KNUs/KNOTs
6. **Open NCRs** — Active nonconformances and age

## Output Formats

- HTML dashboard (auto-generated)
- CSV export for programme reporting
