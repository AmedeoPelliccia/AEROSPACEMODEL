## Purpose

Defines the HAPSSE_BASE product variant: the baseline solar-electric High
Altitude Pseudo-Satellite configuration for persistent stratospheric operations.

## Vehicle Description

Ultra-lightweight solar-electric unmanned aerial system designed for autonomous
station-keeping at 18–22 km altitude. Solar array and rechargeable battery
architecture sized for year-round operation at target latitudes including
winter-solstice energy minimum.

### Key Characteristics

| Parameter                  | Target                                        |
|----------------------------|-----------------------------------------------|
| Operating altitude         | 18–22 km (lower stratosphere)                 |
| Endurance                  | Multi-month continuous (target: 6+ months)    |
| Wingspan                   | 30–80 m (configuration-dependent)             |
| Propulsion                 | Solar-electric (distributed propellers)        |
| Energy storage             | Li-S or Li-ion rechargeable batteries          |
| Payload capacity           | 5–50 kg (mission-dependent)                   |
| Launch/recovery            | Autonomous runway or hand-launch               |
| Regulatory framework       | EASA Light UAS / Special Condition              |
