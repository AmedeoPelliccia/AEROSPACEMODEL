## Problem Statement

The AMPEL360 Q100 BWB programme addresses the need for a net-zero capable 100-passenger transport aircraft leveraging hydrogen fuel cell propulsion and blended wing body aerodynamics. The programme inherits a shared technology base from the AAM/UAM eVTOL, enabling KNOT spillover (coefficient A = 0.50) to accelerate technology maturation across both programmes.

## Key Design Drivers

- **Net-zero propulsion:** Hydrogen Fuel Cell (H₂ FC) primary power with distributed electric open-fan propulsion
- **Aerodynamic efficiency:** Blended Wing Body configuration targeting L/D > 25
- **Technology inheritance:** Shared battery buffer management, motor controller assurance (DO-178C/DO-254), and certification pathway precedent from AAM/UAM SC-VTOL
- **Certification:** CS-25 + Special Conditions (EASA, 2030+ target) leveraging SC-VTOL precedent
