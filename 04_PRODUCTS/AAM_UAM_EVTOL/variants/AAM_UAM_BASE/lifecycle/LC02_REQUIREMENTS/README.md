## Mission Definition Requirements

This phase captures the quantified mission definition parameters derived from the operational concept (LC01). All parameters define the network-level system boundary, not just the vehicle.

### Mission Definition Parameters

| Parameter        | Value                                                                 |
|------------------|-----------------------------------------------------------------------|
| Mission class    | Covers altitudes across two altitude strata (ground → 300 m AGL)     |
| Design range     | 30–150 km (metro / regional)                                          |
| Cruise speed     | 150–250 km/h TAS                                                      |
| Cruise altitude  | 150–300 m AGL (U-space class UBL)                                     |
| VTOL performance | Vertical to/from, ≤80 dB(A) at vertiport boundary                    |
| Passengers       | Single pilot or autonomous; 1–6 pax                                  |
| MTOM range       | 700–3,175 kg (SC-VTOL compliant)                                      |
| Payload          | 80–600 kg (pax + baggage)                                             |
| Reserves         | 20 min VFR hold + diversion to alternate (current rules)             |
| Dispatch target  | 10 flights/day per vehicle                                            |
| Weather minima   | Initially VFR-IFR with maturity (SC-VTOL Cat 1a)                     |
| Infra footprint  | Vertiport ≥ 1.5 D pad (D = 10 × FATO outer envelope)                |

### Mission Profile Tabs

| Tab              | Scope                                                                 |
|------------------|-----------------------------------------------------------------------|
| Mission def.     | Range, altitude, speed, VTOL performance, MTOM, payload, reserves    |
| Energy           | Battery capacity, charging time, C-rate, SoH management              |
| Propulsion       | Rotor count (4–18), tilt architecture, motor power, redundancy        |
| Operations       | Dispatch rate, turnaround, UTM integration, weather minima            |
| Certification    | SC-VTOL basis, DAL allocations, safety objectives                     |
| KNOTs            | Governing control nodes and compliance status                         |
